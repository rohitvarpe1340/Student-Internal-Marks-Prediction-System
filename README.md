# Student-Internal-Marks-Prediction-System

## Tech Stack
Angular | Node.js | MySQL | C++

## Description
A full-stack application to predict students' internal marks using C++ ML logic
integrated with Node.js backend and Angular frontend.

## Folder Structure
- frontend: Angular application
- backend: Node.js REST APIs
- cpp-ml: C++ prediction logic
- database: MySQL schema

## How It Works
1. User enters student data in Angular UI
2. Node.js API calls C++ logic using child process
3. Predicted marks returned and displayed
